# AEGeAn Toolkit

AEGeAn: <b>a</b>nalysis and <b>e</b>valuation of <b>ge</b>nome <b>an</b>notations

The AEGeAn Toolkit provides a bundle of tools for managing and analyzing whole-genome gene structure annotations.
