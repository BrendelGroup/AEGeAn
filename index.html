Integrated genome analysis toolkit

The AEGeAn Toolkit is designed for the Analysis and Evaluation of Genome Annotations. The toolkit includes a variety of analysis programs as well as a C library whose API provides access to AEGeAn's core functions and data structures. AEGeAn is free to use and is released under an unrestrictive open source license.

Tools

The AEGeAn Toolkit started as several distinct but related efforts to develop tools for managing and analyzing whole-genome gene structure annotations. AEGeAn brings these efforts together into one integrated package.

  • CanonGFF3 is a preprocessing tool for GFF3 files—it singles out features related to canonical protein-coding genes and validates the gene models annotated by these features.
  • LocusPocus computes gene loci based on overlapping gene models from one or more sources of annotation.
  • ParsEval compares gene structure annotations from two alternative sources of annotation for the same sequence.
  • VAnG validates a GFF3 file against a schema (in the early stages of development).

Several additional related tools are under development, and will be integrated and documented when they are a bit more stable.

Documentation

Comprehensive documentation for the AEGeAn Toolkit is coming soon! In the mean time, installation instructions are available with the source code distribution, each program provides a descriptive usage statement, and the C API is documented in the header files located in the inc/core directory.
